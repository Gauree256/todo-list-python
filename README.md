# 📝 To-Do List Application (Python)

A simple command-line To-Do List app built with Python that lets you add, delete, and display tasks with deadlines.

## Features

- ✅ Add tasks with a deadline (DD-MM-YYYY format)
- 🗑️ Delete tasks by number
- 📋 Display all current tasks
- 🔒 Input validation for date format

## How to Run

Make sure you have Python 3 installed.

```bash
python todo_list.py
```

## Usage
1.Add Task
2.Delete Task
3.Display Tasks
4.Exit

## Requirements

- Python 3.x
- No external libraries needed
